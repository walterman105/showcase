Showcase

A pure-software wireless CarPlay receiver for jailbroken iPads.

Compatibility

  • Rootful build for iOS 12-14, packaged as iphoneos-arm.
  • Rootless build for iOS 15-17 on Dopamine and palera1n rootless, packaged as iphoneos-arm64.
  • Tested by the author on iPad Air 1 cellular, iOS 12.5.8, checkra1n.
  • Requires Personal Hotspot support through an active SIM or carrier provisioning.

Dependencies

  • Bundled BTstack runtime files.
  • Bundled BTstack launchd socket plist for /tmp/BTstack.
  • OpenSSL libssl3 from Procursus.
  • uikittools for SpringBoard icon registration.

Notes

Showcase takes over classic Bluetooth while active and restores Apple's bluetoothd when stopped. Beta 2-8 fixes a rootless Bluetooth authentication failure seen after SDP on some tester devices.

Changelog

1.0 beta 2-8

  • Changes the bundled rootless BTdaemon SSP persona to DisplayYesNo with dedicated bonding.
  • Targets the Bluetooth authentication failure seen after SDP, where the iPhone returned Simple Pairing Complete status 0x05 before RFCOMM opened.
  • Keeps BTdaemon auto-accept enabled, so testers do not need to type a pairing code.
  • Improves Send Log package-state capture by using dpkg-query directly on rootless systems.
  • Keeps the iOS 12-14 rootful package and the iOS 15-17 rootless package on beta 2-8.

1.0 beta 2-7

  • Adds Send Log in the info sheet with a shareable tar diagnostics archive.
  • Includes Showcase logs, BTstack HCI dumps, package state, process list, interfaces, launchd state, and install metadata.
  • Enables BTstack PacketLogger capture by default before the Bluetooth daemon starts.
  • Leaves rootless HCI accept, IO-capability, and confirmation replies to BTdaemon on Dopamine.
  • Adds clearer Bluetooth logs for disconnect reasons, pairing events, encryption state, RFCOMM, and SDP bring-up.

1.0 beta 2-6

  • Official Dopamine rootless compatibility for iOS 15-17.
  • Native iPhone receiver UI with fullscreen video and pinch-revealed controls.
  • Improved session resilience after short app switches and idle encrypted AirPlay reads.
