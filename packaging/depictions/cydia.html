Showcase

A pure-software wireless CarPlay receiver for jailbroken iPads.

Compatibility

  • Rootful build for iOS 12-14, packaged as iphoneos-arm.
  • Rootless build for iOS 15-17 on Dopamine and palera1n rootless, packaged as iphoneos-arm64.
  • Tested by the author on iPad Air 1 cellular, iOS 12.5.8, checkra1n.
  • Requires Personal Hotspot support through an active SIM or carrier provisioning.

Dependencies

  • Bundled BTstack runtime files.
  • Bundled BTstack launchd socket plist for /tmp/BTstack.
  • OpenSSL libssl3 from Procursus.
  • uikittools for SpringBoard icon registration.

Notes

Showcase takes over classic Bluetooth while active and restores Apple's bluetoothd when stopped. Beta 2-14 sends a fresh post-transport 0x5703 after 0x4E0E so Wi-Fi join can arm the CarPlay network phase.

Changelog

1.0 beta 2-14

  • Separates pre-transport and post-transport Wi-Fi configuration state.
  • Sends a fresh POST-TRANSPORT 0x5703 immediately after 0x4E0E DeviceTransportIdentifierNotification.
  • Still sends a PRE-TRANSPORT compatibility 0x5703 for early 0x5702 requests.
  • Keeps 0x4301 disabled, active multicast mDNS announcements, and bridge100 tcpdump capture.
  • Rootless package only for this release; rootful stays on beta 2-8.

1.0 beta 2-13

  • Sends 0x5703 AccessoryWiFiConfigurationInformation only once per RFCOMM/iAP2 session.
  • Suppresses duplicate 0x5702 Wi-Fi config requests after the first 0x5703 has been sent.
  • Adds state logs around 0x5702, 0x4E0D, 0x4E0E, and duplicate suppression for tester pcaps.
  • Keeps 0x4301 disabled and keeps the beta 2-12 active multicast mDNS announcer.
  • Rootless package only for this release; rootful stays on beta 2-8.

1.0 beta 2-12

  • Sends active multicast mDNS announcements for _airplay._tcp and _raop._tcp on 224.0.0.251:5353 during the handoff window.
  • Includes PTR, SRV, TXT, and A records for the receiver, with the bridge100 address advertised as the host address.
  • Keeps the beta 2-11 bridge100 packet capture and Send Network Dump workflow.
  • Targets senders that join Personal Hotspot but do not actively query _airplay._tcp afterward.
  • Rootless package only for this release; rootful stays on beta 2-8.

1.0 beta 2-11

  • Re-announces _airplay._tcp and _raop._tcp TXT records every 3 seconds for 5 minutes after network services start.
  • Starts a bridge100 pcap automatically when the app reaches Connect from your iPhone.
  • Adds Send Network Dump in the info sheet, separate from the normal diagnostics tar.
  • Adds tcpdump as a rootless dependency so packet capture is available after installation.
  • Rootless package only for this release; rootful stays on beta 2-8.

1.0 beta 2-10

  • Disables the undeclared 0x4301 CarPlayStartSession path by default on rootless builds.
  • Keeps the standard 0x5703 Wi-Fi provisioning plus Bonjour/AirPlay handoff path.
  • Targets the post-Wi-Fi-provisioning failure where the iPhone never appears on bridge100 and never connects to AirPlay port 7000.
  • Preserves the beta 2-9 rootless BTstack Secure Connections fix.
  • Rootless package only for this release; rootful stays on beta 2-8.

  • Disables Secure Connections in the bundled rootless BTstack daemon.
  • Forces legacy SSP pairing for tested Bluetooth controllers.

  • Targets the Bluetooth authentication failure seen after SDP, where the iPhone returned Simple Pairing Complete status 0x05 before RFCOMM opened.
